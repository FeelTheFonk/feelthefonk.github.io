# Niveau 4 : Les Miroirs

## Description
Une salle de miroirs interconnectés où la réalité se déforme. Les joueurs doivent décoder des messages réfléchis et découvrir une séquence cachée basée sur des concepts mathématiques.

## Éléments visuels
- Six miroirs disposés en hexagone
- Reflets dynamiques avec effets de distorsion
- Messages codés en hexadécimal
- Console de debug interactive
- Fragments de vérité cachés

## Vulnérabilités & Solutions

### 1. Séquence Pi
Les miroirs doivent être activés dans l'ordre suivant les premiers chiffres de π :
```
3, 1, 4, 1
```

### 2. Fragments de vérité
Messages codés en hexadécimal dissimulés :
```
4E 6F 74 = "Not"
68 69 6E 67 = "hing"
69 73 = "is"
72 65 61 6C = "real"
```
Message complet : "Nothing is real"

### 3. Easter Eggs
- Commande secrète "mirror" : révèle un indice sur π
- Commande secrète "truth" : décode les fragments
- Séquence spéciale [3,1,4,1,5,9] : révèle tous les fragments

### 4. Console de Debug
Terminal interactif qui :
- Affiche les messages système
- Réagit aux commandes secrètes
- Fournit des retours sur les actions
- Cache des indices dans les messages d'erreur

### Solution complète
1. Observer la disposition hexagonale des miroirs
2. Découvrir les commandes secrètes via exploration
3. Décoder les fragments hexadécimaux
4. Comprendre la référence à π
5. Activer les miroirs dans l'ordre : 3-1-4-1
6. Observer la console pour validation

### Apprentissages techniques
- Manipulation DOM avancée
- Transformations 3D CSS
- Encodage hexadécimal
- Gestion d'événements complexe
- Classes ES6
- Motifs de conception (Observer)
- Animations CSS transitions
- Manipulation de séquences
